## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/luisotavio756/manual-react-app.git

# Go into the repository
$ cd manual-react-app

# Install packages
$ yarn install or npm install

# Run start
$ yarn dev or npm run dev

# Enjoy!
```
